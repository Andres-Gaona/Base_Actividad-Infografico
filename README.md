# Panel-Infografico
Aplicación web infográfica desarrollada en PHP, CSS y JavaScript, diseñada para pantallas informativas.
El sistema presenta contenido dinámico distribuido en módulos independientes —carruseles de imágenes para actividades y shows, listas informativas, reproductor de video, barra superior y ticker inferior— cada uno con estilos, animaciones y lógica propios. La arquitectura separa contenido, lógica y presentación, permitiendo reorganizar el layout visual sin alterar datos, gráficos, animaciones ni funcionalidad interna.
